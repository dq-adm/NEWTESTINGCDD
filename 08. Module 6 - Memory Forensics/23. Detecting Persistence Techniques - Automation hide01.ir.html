(2) Volatility winesap plugin

Winesap is a third-party plugin that can help you automate inspecting persistence-related registry keys. Instead of manually checking keys one by one using the printkey plugin, Winesap will aggregate data from several known persistence-related keys, streamlining the analysis exercise.

To use the winesap keys plugin, run the following command:

volatility -f <memory_dump> --profile=<profile> -g 0xf803788a44d8 winesap
 

Winesap output 1/2 - zoom in

Winesap output 2/2 - zoom in

 

As you can see, it has listed all existing services and the values of other important keys. One cool thing about Winesap is that you can use the '--match' parameter to display suspicious entries.


Notes:

  • You can inspect the same keys on a live system using Sysinternals Autoruns.
  • Although attackers always prefer HKLM keys because they apply to all users, remember to inspect other user-specific variants:
    • HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Run
    • HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run
  • Malware can use multiple keys to persist.
