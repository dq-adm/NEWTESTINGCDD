URLs Disarm/Rewrite

The rewrite feature is an SEG option that neutralizes hyperlinks before they reach the user inbox. It makes hyperlinks readable but non-clickable using multiple ways, such as replacing 'HTTP' with 'hxxp' at the beginning of the link. This way, the user will have to copy the link manually, paste it into the browser and replace/remove 'hxxp'. Such action forces the user to quickly scan the actual URL being visited regardless of what is displayed in the mail body.
 

An attacker may include the following hyperlink in a phishing mail body; https://mail.corporate.com/, which redirects to https://evil.com and not https://mail.corporate.com as the user sees in the mail body.
