Network-Based Intrusion and Prevention Systems (NIDS/NIPS)

An intrusion is an act of wrongfully entering upon, seizing, or taking possession of the property of another. In cyber security, a security intrusion -often referred to as a breach- is unauthorized access to company assets with malicious intent. Intrusion detection/prevention systems play a massive role in identifying and stopping attacks. They come in two types; network-based intrusion detection systems (NIDS) and host-based intrusion detection systems (HIDS).

One may ask, what is the difference between a firewall and an IPS? A firewall inspects network information such as IP address, network port, and protocol and makes decisions based on the rules defined in the firewall policy. On the other hand, an IPS inspects the content of the request and detects/blocks threats based either on behavior analysis or through the use of attack signatures and patterns.

An attacker can use an allowed port and protocol to launch an attack that will make it through the firewall, and that's where the IPS comes in handy. The IPS inspects the content regardless of the network IP, port, or protocol. Nowadays, both functions exist in a single security solution, such as next-generation firewalls that offer firewall and IPS capabilities.

While an IPS can detect and prevent attacks, an IDS detects only without blocking. Intrusion detection systems can be either software you run on a Linux server -such as Suricata- or a hardware security solution/device you need to install on your network -such as McAfee NIPS.

One of the most-popular NIDS is Suricata. Suricata is an open-source IDS developed by the Open Information Security Foundation (OISF). Its official website is https://suricata.io/.

Another popular network intrusion detection system is Zeek.
 

Zeek (formerly Bro) is the world's leading platform for network security monitoring. Flexible, open-source, and powered by defenders. Zeek is a passive, open-source network traffic analyzer. Many operators use Zeek as a network security monitor (NSM) to support investigations of suspicious or malicious activity. Zeek also supports a wide range of traffic analysis tasks beyond the security domain, including performance measurement and troubleshooting.

Zeek provides an online playground where you can learn how to develop Zeek scripts. You can explore it at: https://try.zeek.org/.
