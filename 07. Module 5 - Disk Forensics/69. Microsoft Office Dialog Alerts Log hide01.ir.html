(10) Microsoft Office alerts

Windows generates a custom event log called "OAlerts.evtx" that contains the text displayed to users in dialogs by Microsoft Office suite applications, such as when saving a file or deleting all emails in Outlook. As many incidents nowadays start with a malicious document, examining this log can help you identify the initial access of a threat actor.
