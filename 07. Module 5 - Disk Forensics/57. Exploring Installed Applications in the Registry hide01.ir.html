(2) Registry

  • Another place that holds information about installed applications is the registry. Here are a list of important keys.

    1. HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall: This key contains information about installed applications, including the display name, publisher, and installation location.

    2. HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Windows\CurrentVersion\Uninstall: This key is similar to the previous one, but it contains information about applications installed for 64-bit systems.

    3. HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\App Paths: This key contains the path to the executable for each installed application.
