(7) BAM & DAM Registry key

Background Activity Moderator (BAM) is a service in Windows that controls the activity of background applications. It was introduced in Windows 10 after the Fall Creators Update (version 1709). BAM provides information about the executable files that were run on the system, including the files' full path, the last execution date and time, and other relevant details.

According to Microsoft, background applications don't have to be non-graphical user interface (GUI) applications. They can be any type of app that can continue to run and perform tasks even when you're not using the app or interacting with its window

The information provided by BAM can be helpful in identifying executables that run without the user's knowledge. This information is stored in the SYSTEM hive under the "ControlSet001\Services\bam\State\UserSettings" key, with subkeys named after the users' SIDs under which the application runs. By examining this key, you can gain insights into the background activity on the system.
