(2) setupapi.dev.log

Besides the registry, Windows has other log files containing connected devices' information. One is the "SetupAPI Device Installation Log Entries," a plaintext log file located at C:\Windows\inf\setupapi.dev.log containing information about Plug and Play devices and driver installation.

Using this log file, you can identify the first time a device was connected to the system. To do so, open the log file using a text editor of your choice and search for the device serial number you extracted earlier from the registry.
